# 📝 Angular Reactive Forms - User Registration System
A simple Angular project built to practice **Reactive Forms**. This application allows users to enter employee information through a registration form and displays the submitted data in a table.

---

## 📌 Project Objective

The purpose of this project is to learn and practice:

- Angular Reactive Forms
- FormGroup & FormControl
- Form Validation
- Handling Form Submission
- Displaying Data in a Table
- Angular Control Flow (`@for`)

---

## 🚀 Features

- User Registration Form
- Full Name Input
- Email Validation
- Phone Number Validation
- Gender Selection
- Role Selection
- Department Selection
- Date of Joining
- Salary Input
- Employment Status
- Address Field
- Terms & Conditions Checkbox
- Display Submitted Users in Table
- Form Reset After Submission

## 📋 Form Fields

| Field | Validation |
|--------|------------|
| Full Name | Required, Minimum 3 Characters |
| Email | Required, Valid Email |
| Phone Number | Required, 10 Digits |
| Gender | Required |
| Role | Required |
| Department | Required |
| Date of Joining | Required |
| Salary | Required, Minimum 0 |
| Status | Required |
| Address | Optional |
| Terms & Conditions | Checkbox |

---


## ⚙️ Reactive Form Structure

The form is created using:

- FormGroup
- FormControl
- Validators

Example validators used:

- Validators.required
- Validators.email
- Validators.minLength()
- Validators.pattern()
- Validators.min()

---

## 📥 Form Submission

When the **Add User** button is clicked:

1. Form values are collected.
2. Data is stored in the `userData` array.
3. Submitted users appear in the table.
4. The form is reset.
