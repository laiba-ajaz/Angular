# Expense Tracker (Angular Template-Driven Forms)

## Project Overview

Expense Tracker is a simple Angular application developed using **Template-Driven Forms**. It allows users to record daily expenses by entering expense details through a user-friendly form. All added expenses are displayed in a table.

This project demonstrates the implementation of Angular's Template-Driven Forms, two-way data binding using `ngModel`, event binding, and basic CRUD concepts using an in-memory array.

---

## Features

- Add a new expense
- Reset/Clear form fields
- Display all expenses in a table
- Show total number of expenses
- Responsive Bootstrap UI
- Angular Template-Driven Forms
- Two-way data binding (`ngModel`)

---


## How It Works

### Add Expense

- User enters expense details.
- Clicking the **Add Expense** button stores the expense in an array.
- The form is automatically cleared after adding the expense.

### Reset Form

- Clicking the **Reset** button clears all input fields.

### Display Expenses

- All added expenses are displayed in a responsive Bootstrap table.
- The total number of expenses is shown above the table.
